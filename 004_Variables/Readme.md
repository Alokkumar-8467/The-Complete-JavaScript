# Variables in JavaScript
## A JavaScript variable is just a name of storage location.
And we can give that name in three ways.

1. Using let keyword.
2. Using const keyword.
3. Using var keyword.
## Rules for naming variables.
A variable name can contain any of the following charaters but must not start with a digit.

1. Capital Letters: A to Z
2. Small Letters: a to z
3. Digits: 0 to 9 (but we can't start variable name with digits)
4. Only two special charaters that are: Dollor "$" and Underscore "_"
